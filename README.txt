Le projet esup-webengine-intranets est un module Nuxeo WebEngine.

Le projet esup-webengine-intranets propose de présenter une section
d'esup-ecm à la manière d'un intranet.

Il permet de :

- naviguer dans les sous-dossiers et les fichiers de la section
- rechercher un document dans la section
- d'afficher l'arborescence de la section
- afficher les derniers documents modifiés ou ajoutés

La section est définie dans le fichier "src/main/resources/OSGI-INF/intranets.properties"
par ex :

section.path=/default-domain/sections/intranets/
